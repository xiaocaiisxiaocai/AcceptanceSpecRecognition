# Text Processing Capability

## ADDED Requirements

### Requirement: 简繁体转换预处理
系统必须（SHALL）支持中文简繁体转换预处理功能，使用OpenCCDotNet库实现。

#### Scenario: 启用简体转台湾繁体
- **GIVEN** 用户在配置中启用"简繁转换"功能
- **AND** 选择转换模式为"简体→台湾繁体"
- **WHEN** 系统进行文本匹配前预处理
- **THEN** 系统调用ZhConverter.HansToTW()方法
- **AND** 将查询文本和数据库文本统一转换为台湾繁体
- **AND** 使用转换后的文本进行匹配

#### Scenario: 启用台湾繁体转简体
- **GIVEN** 用户在配置中启用"简繁转换"功能
- **AND** 选择转换模式为"台湾繁体→简体"
- **WHEN** 系统进行文本匹配前预处理
- **THEN** 系统调用ZhConverter.TWToHans()方法
- **AND** 将查询文本和数据库文本统一转换为简体中文
- **AND** 使用转换后的文本进行匹配

#### Scenario: 禁用简繁转换
- **GIVEN** 用户在配置中禁用"简繁转换"功能
- **WHEN** 系统进行文本匹配
- **THEN** 系统不进行简繁转换
- **AND** 直接使用原始文本进行匹配

#### Scenario: 简繁转换初始化
- **GIVEN** 应用程序启动
- **WHEN** 系统初始化文本处理服务
- **THEN** 系统调用ZhConverter.Initialize()初始化字典
- **AND** 初始化仅执行一次

#### Scenario: 支持的转换模式
- **GIVEN** 用户配置简繁转换
- **WHEN** 用户选择转换模式
- **THEN** 系统提供以下选项：
  - 简体→台湾繁体 (HansToTW)
  - 台湾繁体→简体 (TWToHans)

### Requirement: 同义词管理
系统必须（SHALL）支持同义词管理功能，用于增强相似度匹配和Embedding匹配的准确性。

#### Scenario: 添加同义词组
- **GIVEN** 用户进入同义词管理界面
- **WHEN** 用户添加同义词组，如"不锈钢,不鏽鋼,SUS,Stainless Steel"
- **THEN** 系统将这些词语存储为一个同义词组
- **AND** 组内任意词语视为语义等价

#### Scenario: 编辑同义词组
- **GIVEN** 已存在同义词组"不锈钢,不鏽鋼,SUS"
- **WHEN** 用户编辑该组，添加"Stainless Steel"
- **THEN** 系统更新同义词组
- **AND** 记录更新时间

#### Scenario: 删除同义词组
- **GIVEN** 已存在同义词组
- **WHEN** 用户删除该组
- **THEN** 系统删除同义词组
- **AND** 后续匹配不再使用该组

#### Scenario: 同义词匹配增强（相似度模式）
- **GIVEN** 用户启用同义词功能
- **AND** 存在同义词组"不锈钢,SUS"
- **WHEN** 查询文本包含"SUS304"，数据库文本包含"不锈钢304"
- **THEN** 系统识别"SUS"和"不锈钢"为同义词
- **AND** 在相似度计算时将其视为匹配
- **AND** 提高该结果的相似度得分

#### Scenario: 同义词匹配增强（Embedding模式）
- **GIVEN** 用户启用同义词功能
- **AND** 存在同义词组
- **WHEN** 系统生成Embedding向量前
- **THEN** 系统将文本中的同义词替换为标准词（组内第一个词）
- **AND** 使用替换后的文本生成向量

#### Scenario: 禁用同义词功能
- **GIVEN** 用户在配置中禁用同义词功能
- **WHEN** 系统进行匹配
- **THEN** 系统不进行同义词替换
- **AND** 直接使用原始文本匹配

#### Scenario: 同义词导入导出
- **GIVEN** 用户在同义词管理界面
- **WHEN** 用户点击导出
- **THEN** 系统将所有同义词组导出为CSV文件
- **AND** 格式为每行一个同义词组，逗号分隔

#### Scenario: 同义词批量导入
- **GIVEN** 用户准备了CSV格式的同义词文件
- **WHEN** 用户导入该文件
- **THEN** 系统解析CSV文件
- **AND** 批量添加同义词组
- **AND** 与已有组合并或提示冲突

### Requirement: 关键字管理
系统必须（SHALL）支持关键字管理功能，用于在填充Word文档时高亮显示指定关键字。

#### Scenario: 添加关键字
- **GIVEN** 用户进入关键字管理界面
- **WHEN** 用户添加关键字"危险"、"禁止"、"注意"
- **THEN** 系统将这些关键字存储到数据库
- **AND** 关键字不区分大小写

#### Scenario: 删除关键字
- **GIVEN** 已存在关键字"危险"
- **WHEN** 用户删除该关键字
- **THEN** 系统从数据库删除
- **AND** 后续填充不再高亮该词

#### Scenario: 配置高亮颜色
- **GIVEN** 用户进入关键字设置
- **WHEN** 用户选择高亮颜色
- **THEN** 系统保存颜色配置（RGB值）
- **AND** 默认颜色为黄色(#FFFF00)

#### Scenario: 启用关键字高亮
- **GIVEN** 用户启用关键字高亮功能
- **AND** 存在关键字"危险"
- **WHEN** 系统填充【验收】或【备注】列
- **AND** 填充内容包含"危险"
- **THEN** 系统在Word文档中将"危险"设置为高亮显示
- **AND** 使用配置的高亮颜色

#### Scenario: 禁用关键字高亮
- **GIVEN** 用户禁用关键字高亮功能
- **WHEN** 系统填充Word文档
- **THEN** 系统不进行关键字高亮处理
- **AND** 直接填充原始文本

#### Scenario: 关键字批量导入
- **GIVEN** 用户准备了关键字列表文件（每行一个）
- **WHEN** 用户导入该文件
- **THEN** 系统批量添加关键字
- **AND** 自动去重

### Requirement: OK/NG格式标准化
系统必须（SHALL）支持将【验收】列中的OK/NG标记转换为文档标准格式。

#### Scenario: 识别OK/NG变体
- **GIVEN** 【验收】列数据可能包含以下变体：
  - "OK"、"ok"、"Ok"
  - "OK✓"、"OK☑"、"OK⬛"、"OK■"
  - "NG"、"ng"、"Ng"
  - "NG✗"、"NG☒"、"NG⬛"、"NG■"
- **WHEN** 系统读取验收数据
- **THEN** 系统识别这些变体为OK或NG标记

#### Scenario: 转换为标准格式
- **GIVEN** 用户启用OK/NG格式转换
- **AND** 配置标准格式为"OK"和"NG"
- **WHEN** 系统填充【验收】列
- **AND** 源数据为"OK☑ 符合标准"
- **THEN** 系统转换为"OK 符合标准"
- **AND** 保留"符合标准"附加文字

#### Scenario: 保留附加文字
- **GIVEN** 【验收】列数据为"OK⬛ 需要复检，注意温度"
- **WHEN** 系统进行格式转换
- **THEN** 系统转换OK标记为标准格式
- **AND** 保留"需要复检，注意温度"文字
- **AND** 最终结果为"OK 需要复检，注意温度"

#### Scenario: 配置标准格式
- **GIVEN** 用户进入OK/NG格式设置
- **WHEN** 用户配置标准格式
- **THEN** 用户可以设置：
  - OK的标准显示文本（默认"OK"）
  - NG的标准显示文本（默认"NG"）
  - 是否使用特殊符号（如☑/☒）

#### Scenario: 禁用格式转换
- **GIVEN** 用户禁用OK/NG格式转换
- **WHEN** 系统填充【验收】列
- **THEN** 系统保留原始格式
- **AND** 不进行任何转换

### Requirement: 文本预处理管道
系统必须（SHALL）提供统一的文本预处理管道，按顺序执行各项预处理。

#### Scenario: 预处理执行顺序
- **GIVEN** 用户启用了多项预处理功能
- **WHEN** 系统处理文本
- **THEN** 系统按以下顺序执行：
  1. 简繁转换（如启用）
  2. 同义词替换（如启用，仅用于匹配）
  3. OK/NG格式转换（如启用，仅用于填充）
  4. 关键字高亮（如启用，仅用于填充）

#### Scenario: 预处理配置界面
- **GIVEN** 用户进入文本处理设置
- **WHEN** 界面加载完成
- **THEN** 显示各项预处理功能的开关：
  - 简繁转换（开/关 + 模式选择）
  - 同义词替换（开/关 + 管理入口）
  - OK/NG格式转换（开/关 + 格式设置）
  - 关键字高亮（开/关 + 颜色设置 + 管理入口）

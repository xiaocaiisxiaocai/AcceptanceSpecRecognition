## ADDED Requirements

### Requirement: 列映射规则管理页面
系统 SHALL 提供一个页面用于维护“导入列映射规则”，用于将 Word 表格的列（通过表头/字段名识别）映射到系统的四个目标字段：项目、规格、验收标准、备注。

#### Scenario: 用户查看并编辑规则
- **WHEN** 用户进入“配置管理 → 列映射规则”
- **THEN** 系统展示四个目标字段的规则配置（可匹配表头列表、匹配模式、优先级）
- **AND** 用户可以新增/编辑/删除规则并保存

### Requirement: 导入页自动预填列映射
系统 SHALL 在“导入数据 → 配置映射”步骤中，基于已配置的列映射规则对每个选中表格自动预填【项目列/规格列/验收标准列/备注列】。

#### Scenario: 多表格自动预填
- **WHEN** 用户上传 Word 并选择多个表格进入“配置映射”
- **THEN** 系统对每个表格自动选择最符合规则的列作为映射
- **AND** 用户仍可逐表调整映射列

#### Scenario: 规则未命中
- **WHEN** 某个表格的表头未命中任意规则
- **THEN** 系统提示该表格需要手动选择映射列

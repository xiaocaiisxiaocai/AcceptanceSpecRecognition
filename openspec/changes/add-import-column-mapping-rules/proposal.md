# Change: 为“导入数据”增加列映射规则（自动套用到多表格）

## Why
导入 Word 时，一个文件可能包含十几个表格。当前需要对每个表格逐一设置【项目列/规格列/验收标准列/备注列】映射，重复劳动多、易出错。

## What Changes
- 新增“列映射规则”配置页，用于维护：哪些表头/字段名可匹配到【项目/规格/验收标准/备注】四个目标列。
- 导入数据流程中，选择多个表格后，系统自动根据“列映射规则”对每个表格进行列映射**预填**（用户仍可逐表调整）。
- 规则支持以“表头文本”进行匹配（包含/等于/正则），并支持优先级，避免多候选时选错。

## Impact
- Affected specs:
  - user-interface
  - configuration
  - data-storage（新增配置持久化）
- Affected code (planned):
  - Backend: 新增配置实体/迁移、规则 CRUD API、导入页自动映射读取接口
  - Frontend: 新增配置页面、导入页自动套用逻辑与提示

## Open Questions
1. 规则作用域：仅“全局一套”够不够？还是需要支持“按客户/制程单独一套规则”（覆盖全局）？
2. 匹配依据：仅用表头文本（推荐）还是也支持“固定列序号”/“列范围”？

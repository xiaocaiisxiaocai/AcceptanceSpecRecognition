## MODIFIED Requirements

### Requirement: 数据导入界面支持 Excel（.xlsx）
系统 SHALL 在“数据导入”界面支持上传 Excel（.xlsx）并按工作表配置导入。

#### Scenario: 上传 Excel 并展示工作表列表
- **GIVEN** 用户进入数据导入界面
- **WHEN** 用户上传一个 `.xlsx` 文件
- **THEN** 界面展示该文件的工作表列表（名称、索引、已用区域摘要）
- **AND** 用户可勾选需要导入的工作表（支持多选）

#### Scenario: 为每个工作表配置列序号
- **GIVEN** 用户已选择一个或多个工作表
- **WHEN** 用户进入“配置映射”步骤
- **THEN** 界面为每个工作表提供配置项：
  - 项目列（必填，输入列序号）
  - 规格内容列（必填，输入列序号）
  - 验收标准列（可选）
  - 备注列（可选）
  - 表头行配置（表头起始行/表头行数）
  - 数据起始行

#### Scenario: 配置错误提示
- **GIVEN** 用户将项目列配置为超出范围的列号
- **WHEN** 用户请求预览或进入下一步
- **THEN** 界面显示明确错误提示（指出哪个字段配置越界）

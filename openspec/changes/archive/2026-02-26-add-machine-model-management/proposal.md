# Change: 新增机型管理与验收规格可选制程/机型

## Why
当前系统只有客户与制程两类基础数据，无法维护机型维度；同时验收规格强制要求制程，无法表达“只有客户、制程可空、机型可空”的业务需求。

## What Changes
- 新增“机型管理”基础数据（全局独立），提供增删改查。
- 验收规格关联关系调整为：客户必填，制程可选，机型可选。
- 导入、筛选与匹配配置增加“机型”选择器；当未选择制程或机型时使用空值范围。

## Impact
- Affected specs: user-interface, data-storage, api
- Affected code: 实体/DTO/控制器/仓储/前端页面/路由/导入与匹配流程

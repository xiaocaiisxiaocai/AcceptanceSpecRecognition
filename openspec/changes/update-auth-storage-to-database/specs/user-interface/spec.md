## ADDED Requirements

### Requirement: 系统用户管理页面
系统 SHALL 提供系统用户管理页面，并支持管理员完成账号运维操作。

#### Scenario: 展示系统用户列表
- **WHEN** 管理员进入系统用户管理页面
- **THEN** 页面展示分页用户列表，包含用户名、昵称、角色、权限、启用状态和更新时间

#### Scenario: 新增系统用户
- **WHEN** 管理员在页面填写用户名、密码、角色等信息并提交
- **THEN** 页面调用创建接口并在成功后刷新用户列表

#### Scenario: 编辑系统用户
- **WHEN** 管理员在页面修改用户昵称、角色、权限或启用状态并保存
- **THEN** 页面调用更新接口并在成功后显示最新数据

#### Scenario: 重置用户密码
- **WHEN** 管理员在页面输入新密码并确认
- **THEN** 页面调用重置密码接口并提示操作结果

#### Scenario: 删除或禁用受限用户
- **WHEN** 管理员尝试删除当前登录账号或触发后端保护规则
- **THEN** 页面阻止或提示失败原因，不进行静默失败
